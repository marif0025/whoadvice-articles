# Hedge-trimmer cluster audits

This folder contains reports that evaluate more than one article or the cluster as a whole.

- `article-generation-hierarchy-audit.md` — parent/leaf intent and generation-readiness review for the original commercial hub and corded/cordless leaves.
- `cluster-audit.md` — topology, shared-product consistency, destination governance, and implementation plan.
- `final-article-audit-2026-07-15.md` — dated multi-article editorial-readiness audit.

Keep package-specific `audit.md` files in their corresponding article folders. New dated cluster audits should use an ISO date in the filename; durable cluster rules belong in `../README.md` or `../content-strategy.md` instead of an audit report.
